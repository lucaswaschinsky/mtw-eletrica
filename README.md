# MTW Elétrica — Landing Page

Landing page para a **MTW Elétrica**, empresa de instalações e manutenção elétrica residencial e comercial na Baixada Santista (Guarujá e Santos).

## Sobre o projeto

Site estático de uma página (`index.html`) desenvolvido em HTML/CSS puro, sem dependências de frameworks ou build tools. O design utiliza tema escuro com destaque em amarelo elétrico.

## Estrutura

```
mtw-eletrica/
└── index.html   # Página única com todo o HTML, CSS e SVGs inline
```

## Seções da página

- **Hero** — apresentação da empresa com CTA para WhatsApp e estatísticas (100+ clientes, 5+ anos de experiência)
- **Serviços** — 6 cards: Instalações Residenciais, Instalações Comerciais, Manutenção Elétrica, Quadro de Distribuição, Iluminação e Segurança Elétrica
- **Por que nos escolher** — diferenciais da empresa (profissional certificado, atendimento rápido, orçamento sem compromisso, garantia no serviço)
- **Contato** — CTA final com link direto para WhatsApp
- **Footer** — links para Instagram e WhatsApp

Abra o `index.html` diretamente no navegador — não requer servidor ou instalação.
